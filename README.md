# rocq-vine-boom README

Simple extension to help you dramatize your proof scripts.

## Features

If you fail an `apply`, `destruct`, reference a value nonexistent in context or provide an incorrect type, prepare to get boomed.

## Requirements

ffmpeg, I guess?

## Known Issues

Does not replay when the same error appears twice (for example, when trying to use the same failing tactic twice).
